# TCF Pricing Tool (RocksMax Version)

This application is a specialized pricing tool for fan equipment with the following features:
- Project-based fan specification and pricing
- Support for different fan models, sizes, and arrangements
- Calculation of fabrication and bought-out component costs
- Project summary with detailed breakdown of costs
- Database integration for storing and retrieving projects
